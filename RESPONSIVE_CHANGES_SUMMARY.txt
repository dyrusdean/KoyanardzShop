================================================================================
CSS TECHNIQUES USED
================================================================================

1. CSS GRID for Card Layout:
   .products-table tbody tr {
       display: grid;
       grid-template-columns: 1fr;
       gap: 0.8rem;
       padding: 1rem;
       background: white;
       border-radius: 8px;
   }

2. Data Attributes for Labels:
   .products-table tbody td::before {
       content: attr(data-label);
       font-weight: 700;
   }
   → Uses existing HTML: <td data-label="Product Name">

3. Display Contents for Table Structure:
   .products-table tbody {
       display: contents;
   }
   → Removes tbody wrapper, displays rows directly

4. Responsive Button Sizing:
   .action-btn {
       flex: 1;
       min-width: calc(33.33% - 0.2rem);
   }
   → 3 buttons per row with proper gap handling
